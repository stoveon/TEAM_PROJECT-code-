# 프로젝트 소개

### 개발기간 2021.09.23 ~ 2021.11.10 (49일)
> 주제선정(5일)<br>
> 요구사항 정의서(6일)<br>
> 요구사항 명세서(7일)<br>
> 설계(14일)<br>
> 구현 및 통합 테스트(16일)<br>
> 문서화(1일)

### 홈페이지 활동으로 포인트를 적립하고, 이를 통해 물건을 구매할 수 있는 웹 사이트
- 집에서 할 수 있는 취미를 공유할 수 있는 커뮤니티 사이트 입니다.
- 메인 페이지, 게시판, 포인트몰, 회원관리, 기타 기능으로 구성되어 있습니다.
- 회원들만 이용할 수 있는 사이트이며, 매주 인기글과 추천글이 상단에 표시됩니다.

### 주요동작
**사용자**
- 로그인 및 회원가입
- 게시판 (정보, 익명, 공지)
- 포인트몰 상품 구매
- 마이페이지, 출석체크

**관리자**
- 로그인
- 주문관리
- 상품관리
- 회원관리
- 게시글 관리
- 신고 관리
- Q&A 관리
